# Smart Expense Tracker Pro

A modern desktop application developed using Python and Tkinter for managing personal expenses.

## Features

- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Expenses
- Monthly Budget Tracking
- Pie Chart Visualization
- Bar Chart Visualization
- Export to Excel

## Technologies

- Python
- Tkinter
- Pandas
- Matplotlib
- tkcalendar
